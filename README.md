# Detection Engineering Lab

Laboratorio práctico de **Detection Engineering** basado en **MITRE ATT&CK**, enfocado en la emulación de técnicas adversarias y su análisis desde el lado defensivo.

Las técnicas son emuladas utilizando **MITRE CALDERA** y validadas mediante la correlación de múltiples fuentes de telemetría:

- Sysmon  
- PowerShell Operational Logs  
- Telemetría de red  
- SIEM (Wazuh)  

## Objetivos

- Emular técnicas reales de adversarios.
- Evaluar la visibilidad defensiva en el endpoint y el SIEM.
- Diseñar detecciones **basadas en comportamiento** (Sigma).
- Validar detecciones en **Wazuh**.

## Herramientas

- MITRE CALDERA  
- Sysmon  
- PowerShell Logging  
- Wazuh  
- Wireshark  

## Técnicas analizadas

- **T1071.001** – Application Layer Protocol: Web Protocols  
  

> Este repositorio está orientado a fines **defensivos y educativos**, y todo el contenido se ejecuta en entornos de laboratorio controlados.
